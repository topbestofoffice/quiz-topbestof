# Quiz Chandra Prakash Verma

A quiz application with integrated ad system featuring Google Publisher Tag (GPT) rewarded ads.

## Project Structure

```
quiz.Chandra Prakash Verma.com/
├── css/                    # Stylesheets
├── data/                   # Data files (categories, questions)
├── images/                 # Image assets
├── js/                     # JavaScript files
├── guides/                 # Documentation & migration guides
├── examples/               # Demo pages & examples
├── tests/                  # Test files & utilities
├── index.html              # Main quiz page
├── playquiz.html           # Quiz playing page
├── home.html               # Home page
├── category.html           # Category selection
├── profile.html            # User profile
└── README.md               # This file
```

## Key Features

- **GPT Rewarded Ads**: Integrated Google Publisher Tag rewarded ad system
- **Quiz System**: Multi-category quiz with coin rewards
- **Responsive Design**: Mobile-first responsive interface
- **LocalStorage**: Persistent coin tracking
- **Debug Tools**: Comprehensive debugging and testing tools

## Quick Start

1. Open `index.html` in a web browser
2. Use the debug panel (top-left) to test ad functionality
3. Navigate through the quiz system
4. Check browser console for detailed logs

## Documentation

- **Guides**: See `guides/` folder for implementation guides and migration documentation
- **Examples**: Check `examples/` folder for demo pages and usage examples
- **Tests**: Use `tests/` folder for testing and verification tools

## Ad System

The project uses Google Publisher Tag (GPT) for rewarded ads:
- Ad Unit: `/22760191030/reward`
- Implementation follows Google's official patterns
- Comprehensive logging and debugging
- Fallback systems for error handling
